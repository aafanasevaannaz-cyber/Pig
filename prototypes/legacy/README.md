# Legacy prototypes

Сюда постепенно переносятся старые html-файлы, которые раньше лежали в корне репозитория.

## Зачем

- очистить корень проекта
- оставить историю старых прототипов
- не смешивать рабочую версию и прошлые эксперименты

## Правило

Новые рабочие изменения идут только в `app/`.
Старые версии и переходные файлы — только в `prototypes/legacy/`.
