## 🔧 ЕСЛИ НЕ СРАБОТАЛО

### Проблема: "Cannot find index.html"

**Решение:**
```
В Vercel Dashboard:
1. Откройте Settings → General
2. Root Directory: оставьте пусто (/)
3. Сохраните
4. Redeploy → Deploy
```

### Проблема: "Стили не загружаются (CSS пуст)"

**Проверьте пути в index.html:**
```html
<!-- ✅ Правильно -->
<link rel="stylesheet" href="styles/main.css">

<!-- ❌ Неправильно -->
<link rel="stylesheet" href="/styles/main.css">
```

### Проблема: "Звуки не работают"

**AudioContext требует HTTPS:**
```
Vercel автоматически дает HTTPS ✅
Если не работает:
1. Подождите 5 минут (кеш браузера)
2. Очистите cache: Ctrl+Shift+Delete
3. Перезагрузите страницу
```

### Проблема: "Сохранение не работает"

**localStorage требует нормального контекста:**
```
✅ На Vercel работает
✅ На localhost работает
❌ На file:// не работает (открытие напрямую)
```

## ❓ FAQ

**Q: Это бесплатно?**
A: Да! Vercel дает 100 GB/месяц трафика бесплатно. Достаточно для миллионов посещений.

**Q: Могу ли я использовать свой домен?**
A: Да, через Vercel или как угодно. Но Vercel домен проще.

**Q: Что если я удалю репозиторий?**
A: Сайт останется жить на Vercel. Но обновляться не будет.

**Q: Как отключить Vercel?**
A: Удалить в Vercel Dashboard. Или просто не использовать ссылку.

**Q: Данные свинки сохраняются?**
A: Да! localStorage браузера это не трогает. Каждый браузер имеет свою свинку.
