# Assets

Папка для ассетов главного приложения.

## Сюда пойдёт

- `mascot/` — слои и изображения свинки
- `sounds/` — мягкие звуки для антистресс-режимов
- `ui/` — декоративные элементы, если понадобятся

## Правило

Главная версия приложения использует ассеты только отсюда, а не из случайных файлов в корне.
