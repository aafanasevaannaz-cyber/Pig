# Sounds

Сюда складываются звуки главной версии приложения.

## Обязательный минимум

- `purr-soft.mp3`
- `wheek-happy.mp3`
- `munch-soft.mp3`
- `sleepy-breath.mp3`
- `house-rustle.mp3`

## Дополнительно

- `sniff-soft.mp3`
- `ui-soft-tap.mp3`
- вариативные версии с суффиксами `-02`, `-03`

## Правило

Звуки должны быть:

- тихими
- короткими
- мягкими
- без резкой атаки
- без тревожного ощущения

## Не делать

- громкие визги
- бесконечный луп
- резкие кнопочные щелчки
- тяжёлые реалистичные звуки страдания
