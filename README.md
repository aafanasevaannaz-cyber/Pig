# 🐷 Свинка-Дружок: Антистресс Тамагочи

Милый и успокаивающий виртуальный питомец - свинка в стиле тамагочи. Идеальное средство для отдыха и релаксации.

## ✨ Особенности

- **Адаптивный дизайн** - работает на мобильных, планшетах и ПК
- **Мягкая геймплей** - никаких штрафов, только позитив
- **Реальные потребности** - свинка имеет голод, энергию и счастье
- **Автоматические изменения** - питомец развивается самостоятельно
- **Красивые анимации** - плавные переходы и приятные визуальные эффекты

## 🎮 Как Играть

1. **Кормить** (🍎) - снизить голод, сделать свинку счастливой
2. **Поиграть** (🎾) - повысить счастье, потратить энергию
3. **Погладить** (🎀) - показать любовь, увеличить счастье
4. **Спать** (😴) - восстановить энергию

## 📊 Статистика

- **Голод** - растёт со временем (показывает сытость)
- **Счастье** - зависит от внимания и условий
- **Энергия** - восстанавливается во время сна
