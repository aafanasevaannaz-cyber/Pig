# Pig

Антистресс-прототип со свинкой-дружком.

## Что внутри
- `index.html` — структура экрана
- `styles.css` — внешний вид и анимации
- `app.js` — поведение, сцены, звуки, авто/ручной режим
- `config.js` — быстрые настройки, включая вотермарк и заголовок

## Как безопасно менять
1. Вотермарк и название — сначала в `config.js`
2. Цвета, размеры, анимации — только в `styles.css`
3. Логику сцен и кнопок — в `app.js`
4. Разметку экрана — в `index.html`

## Рекомендованный поток
- `main` держать только для рабочей версии
- любую новую идею делать в отдельной ветке
- смотреть diff перед merge
- не мешать стиль, логику и тексты в одном коммите
