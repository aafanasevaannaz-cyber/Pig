# Pig

Свинка-дружок как антистресс‑тамагочи.

## Структура

- `app/` — главная рабочая версия
- `prototypes/` — прототипы и переходные версии
- `docs/` — заметки по логике и состояниям
- `vercel.json` — маршрутизация для Vercel

## Главные пути

- `/app/` — основное приложение
- `/prototypes/live/` — живая прототипная ветка

## Что уже лежит в legacy

Старые html-файлы в корне пока оставлены как legacy-слой. После проверки новой структуры их можно будет перенести или удалить.

## Как безопасно менять

1. Внешность и разметка — сначала в `app/index.html`
2. Стили и анимации — в `app/styles.css`
3. Логика режимов и интерактив — в `app/app.js`
4. Новые эксперименты — только через `prototypes/`
