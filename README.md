# 🐷 Свинка-Дружок: Антистресс Тамагочи

> Милый и успокаивающий виртуальный питомец для релаксации и отдыха

![Version](https://img.shields.io/badge/version-1.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Особенности

- 🎮 **Интерактивная геймплей** - кормить, гладить, играть, спать
- 🎨 **Красивый дизайн** - милая свинка, нарисованная на CSS
- 📱 **Адаптивный** - работает на мобильных, планшетах и ПК
- 🔊 **Звуки** - приятные звуковые эффекты (можно отключить)
- 💾 **Сохранение** - состояние питомца сохраняется в браузере
- 🎯 **Мягкая геймплей** - никаких штрафов, только позитив и забота

## 🎮 Как Играть

### Доступные действия:

- **🍎 Покормить** - дай свинке еду
- **🎀 Погладить** - покажи свинке любовь (кликай на неё!)
- **🎾 Поиграть** - активная игра
- **😴 Спать** - свинка восстанавливает энергию

### Статистика:

| Параметр | Описание |
|----------|---------|
| 🍎 Голод | Растёт со временем (ниже = лучше) |
| 💕 Счастье | Зависит от твоей заботы |
| ⚡ Энергия | Восстанавливается только во сне |

## 🌟 Настроения свинки

- 😴 **Спит** - восстанавливает энергию
- 🤩 **Восторг** - очень счастлива
- 😊 **Довольна** - хорошее состояние
- 🎾 **Игривая** - хочет играть
- 😌 **Спокойна** - отдыхает
- 😋 **Голодна** - хочет кушать
- 😪 **Устала** - нужно поспать

## 🎛️ Управление

- **Звук 🔊** - включить/выключить звуки
- **Сброс ↻** - новая игра
- **Кликай на свинку** - погладить (бесплатное действие!)

## 🛠️ Технологии

- ✅ **Ванильный JavaScript** - без фреймворков
- ✅ **CSS3** - анимации, гриды, флексбоксы
- ✅ **Web Audio API** - генерация звуков
- ✅ **LocalStorage** - сохранение прогресса

## 🚀 Запуск

### Локально:
```bash
# Просто откройте файл в браузере
open index.html
```

### На сервере:
```bash
# Используйте любой простой HTTP сервер
python -m http.server 8000
# или
npx http-server
```

Затем откройте `http://localhost:8000`

## 💡 Философия

Это не игра с целями или концом. Это просто милый компаньон для релаксации.

**Нет наказаний** - свинка не может умереть или заболеть. Она просто становится грустнее.

**Простая механика** - fokus на расслаблении, не на управлении.

**Открытое сердце** - проект создан с любовью ❤️

## 🎨 Кастомизация

Хочешь изменить цвета? Отредактируй переменные в `styles/main.css`:

```css
:root {
  --primary: #FFB6D9;    /* Основной цвет */
  --secondary: #FFE5F0;  /* Вторичный цвет */
  --accent: #FF88CC;     /* Акцент */
  --text: #654321;       /* Цвет текста */
  --bg: #FFF9E6;         /* Цвет фона */
}
```

## 🐛 Известные особенности

- Свинка может быть разбужена только через кнопку "Спать"
- Во время сна действия недоступны
- Счастье падает быстрее, если свинка голодна или устала
- Звуки генерируются в реальном времени без внешних файлов
